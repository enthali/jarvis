## Summary

Add three context-menu actions to project and event leaf nodes in the Jarvis Explorer sidebar:
"Reveal in Explorer" (VS Code file explorer), "Reveal in File Explorer" (OS file manager),
and "Open in Terminal". All three delegate to built-in VS Code commands and require no custom
logic beyond command registration and manifest wiring.
