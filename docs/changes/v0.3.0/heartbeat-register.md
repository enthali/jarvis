## Summary

Unify periodic tasks by adding a job registration API to the heartbeat scheduler.
Extension modules (starting with YamlScanner) register heartbeat jobs instead of
managing their own timers. The `jarvis.scanInterval` setting changes from seconds
to minutes, with 0 = disabled.
